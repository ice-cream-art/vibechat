# VibeChat 提交材料

## 1. 100 字以内产品介绍

VibeChat 是一款 AI 驱动的匿名情绪社交应用。它理解用户此刻的情绪类型、强度与倾向，据此寻找真正“同频”的陌生人，并生成自然的匿名对话空间，让每次连接从被理解开始。

## 2. 3-6 分钟演示视频脚本

### 00:00-00:25 产品概念

“多数社交产品先问你是谁，VibeChat 先理解你此刻感觉如何。我们把 AI 情绪识别真正用于匹配，而不是只贴一个标签。”

### 00:25-01:15 情绪输入与识别

1. 用户 A 输入：“比赛快开始了，我既期待又焦虑，担心自己来不及完成。”
2. 展示主情绪、强度、正负向、关键词、混合情绪和解释。
3. 点明结果来自统一的结构化情绪 Schema。

### 01:15-02:10 自动匹配

1. 用户 A 进入匹配等待页。
2. 无痕窗口中的用户 B 输入相近内容并加入匹配。
3. 展示 90% 以上同频指数及“都带着焦虑、情绪强度接近”的理由。
4. 说明匹配分数直接读取情绪标签、强度、正负向和唤醒度。

### 02:10-03:15 匿名实时聊天

1. 展示系统生成的两个匿名身份。
2. 用户 A/B 各发送两条消息，证明双向实时收发。
3. 展示时间顺序、在线状态和不暴露真实身份的界面。

### 03:15-04:10 技术亮点

1. 展示 `LLM_PROVIDER=openai | anthropic | demo`。
2. 展示 OpenAI Chat Completions 与 Anthropic Messages 共用同一 `EmotionResult`。
3. 展示 FastAPI API 文档、自动测试和 WebSocket 架构。

### 04:10-04:40 兜底与安全

1. 演示等待后出现“同频向导”。
2. 说明 LLM 异常提示、配置隔离和极端负面情绪安全提示。
3. 回到公网地址收尾：“让每次连接，从被理解开始。”
